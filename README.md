# 🗽 New York Airbnb Neighbourhood Market Analysis


## 📌 Project Overview

This project provides a detailed data analysis of Airbnb listings across New York City neighbourhoods. Using the latest dataset, we uncover pricing trends, availability insights, host behaviors, and neighbourhood market strengths.

> 🔍 Objective: Help potential hosts and travelers make informed decisions based on real Airbnb data.

---

## 📂 Dataset

- **Source:** [Inside Airbnb](http://insideairbnb.com)
- **File:** `AB_NYC_2019.csv`
- **Total Records:** ~49,000+
- **Attributes:** Neighbourhood, Room Type, Price, Availability, Reviews, Minimum Nights, etc.

---

## 📊 Key Insights & Analysis

### ✅ Completed Questions
